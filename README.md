📊 Social Media Analysis Dashboard – Power BI Project

📌 Project Overview

This project focuses on analyzing social media usage patterns using Microsoft Power BI. The dashboard provides insights into user behavior, addiction levels, platform preferences, demographic trends, and usage purposes across different social media platforms.

The analysis helps identify:

Most used social media platforms

Addiction level distribution

Daily usage trends by age and gender

Purpose of social media usage

Platform-wise engagement comparison



---

🛠️ Tools & Technologies Used

Microsoft Power BI

Dataset (CSV / Excel)

Data Cleaning & Visualization Techniques

DAX for calculations and measures

The dataset contains information related to:

📥 Importing Dataset into Power BI

Step 1: Open Power BI Desktop

Launch Microsoft Power BI Desktop.

Step 2: Load Dataset

1. Click Home → Get Data


2. Choose:

Excel Workbook / CSV File



3. Browse and select the dataset file.


4. Click Load
